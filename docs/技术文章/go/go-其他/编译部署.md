## windows上项目打包二进制文件部署到linux
1. 设置windows 环境变量，   $env:GOOS="linux" 使用go env 命令确认
2. go build  -o myapp
3. 复制文件到linux 
  >  chmod +x myapp

4. 后台运行
> nohup ./myapp > custom_output.log 2>&1 &
